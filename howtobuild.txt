1. 环境准备
    安装cnpm。使用淘宝的镜像。这样安装会快一点.（不安装后面的cnpm命令都使用npm，可能会慢点）
    > npm install cnpm -g --registry=https://registry.npm.taobao.org
    > cnpm install -g ionic cordova    // 安装 ionic|cordova
    > cnpm install                     // 安装依赖包

2. 编译 Launcher的代码
    https://github.com/elastos/Elastos.Trinity.DApps.Launcher.git  还是原来的feat_launcher分支
    切换到应用根目录
    > cd Elastos.Trinity.DApps.Launcher
    > cnpm install          // 安装依赖
    > ionic build --prod   // 编译
    输入ionic build --prod 会产生一个www文件夹，这个就是launcher的输出文件。

3. 编译 DApps Samples
    > git clone git@github.com:elastos/Elastos.Trinity.DApps.Samples.git
    > cd Elastos.Trinity.DApps.Samples
    > cnpm install
    > ionic build --prod   // 生成www文件夹，这个就是DApps Samples的输出文件。

4. 编译runtime
    4.1 clone runtime代码并切换到dev_app分支
        > git clone git@github.com:elastos/Elastos.Trinity.Runtime.git
        > cd Elastos.Trinity.Runtime
        > git checkout -b dev_app origin/dev_app
        > cordova prepare
